## ぱぱっとごはん

### アプリ名

ぱぱっとごはん

#### コンセプト

迷う前に、近くのごはん

- 現在地を取得して周辺の飲食店を表示
- ジャンルから直感的に探せる
- 「すぐごはん」を見つけられる

## アプリケーション機能

### 機能一覧

- レストラン検索：ホットペッパーグルメサーチ API を使用して、現在地周辺の飲食店を検索する。
- レストラン情報取得：ホットペッパーグルメサーチ API を使用して、飲食店の詳細情報を取得する。
- 検索条件の変更：検索半径、ジャンル、予算の変更
- 店舗一覧の表示：店名・画像・住所
- 店舗詳細の表示：モーダルの表示「画像、住所、営業時間」
- 検索中のローディング表示

### 画面一覧

- 検索画面と一覧画面：条件を指定してレストランを検索、検索結果の飲食店を一覧表示する。
- モーダル画面：画像、住所、営業時間の表示
